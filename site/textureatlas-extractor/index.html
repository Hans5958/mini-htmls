TextureAtlas Extractor

A very simple extractor/unpacker for texture spritesheets. Initially with Starling's format (for Friday Night Funkin'), it is now compatible on multiple formats.

This tool is quite experimental. Please report issues on the issues tab.

Supported formats: cocos2d cocos2d (2) Egret2D Godot JSON (array), Phaser (array) JSON (hash), Phaser (hash), Unreal Engine (Paper2d), PixiJS Phaser 3 Starling UIKit XML (generic)
