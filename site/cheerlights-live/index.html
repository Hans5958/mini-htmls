About CheerLights Live

This page displays the current light on CheerLights, a public light service/"global network of synchronized lights" that everyone can control. You are a part of "the network" by opening this page, and you can control it.

To control the light, simply interact with the bot present on the CheerLights Discord server, or mentioning the account on Mastodon, @CheerLights@botsin.space. The list of colors are present on their website.

This page also serves an educational purpose due to how simple it is (get the color from the API, show the color on the page), has a substantial amount of documentation, and the usage of a public, active API. The source code can be viewed on the GitHub repository.
